== Description ==
1. Show WooCommerce Product Price Per Weight on single product and category page.
2. Show WooCommerce Product Weight on single product and category page.
3. It also works on sale products and it will change the price dynamically when sale date ends.
4. Free plugin adds features compatible with WordPress and WooCommerce.
5. Elements classes for styling are provided too : 
- on WooCommerce single product page :
weight element class : 'ppw_weight_product'
price element class : 'ppw_price_product'
- on WooCommerce single category page :
weight element class : 'ppw_weight_category'
price element class : 'ppw_price_category'
6. FREE PLUGIN - show support rate 5 stars and share :)

== Installation ==
1. install the plugin and activate.
2. main settings page under WooCommerce menu named Price Per Weight.
3. to modify elements styles you can use the class names : 
- on WooCommerce single product page :
* weight element class : 'ppw_weight_product'
* price element class : 'ppw_price_product'
- on WooCommerce single category page :
weight element class : 'ppw_weight_category'
price element class : 'ppw_price_category'
